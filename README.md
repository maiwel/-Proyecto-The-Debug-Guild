# 🎬 The Debug Guild — Catálogo de Películas

Aplicación fullstack para gestionar un catálogo de películas, desarrollada con **Angular** en el frontend y **Spring Boot** en el backend.

---

## 🛠️ Tecnologías

| Capa | Tecnología |
|---|---|
| Frontend | Angular 17+ |
| Backend | Spring Boot 3 |
| Base de datos | H2 (en memoria) |
| Control de versiones | Git + GitHub |
