# 🎬 The Debug Guild — Catálogo de Películas

Aplicación fullstack para gestionar un catálogo de películas, desarrollada con **Angular** en el frontend y **Spring Boot** en el backend.

---

## 🛠️ Tecnologías

| Capa | Tecnología |
|------|------------|
| Frontend | Angular 17+ |
| Backend | Spring Boot 3 |
| Base de datos | H2 (en memoria) |
| Control de versiones | Git + GitHub |

---

## 📁 Estructura del proyecto

```
-Proyecto-The-Debug-Guild/
├── frontend/       # Aplicación Angular
├── backend/        # API REST con Spring Boot
├── .gitignore
├── package-lock.json
└── README.md
```

---

## 🚀 Instalación y ejecución

### Requisitos previos

- Node.js 18+
- Angular CLI 17+
- Java 17+
- Maven

### Frontend

```bash
cd frontend
npm install
ng serve
```

La aplicación estará disponible en `http://localhost:4200`

### Backend

```bash
cd backend
./mvnw spring-boot:run
```

La API estará disponible en `http://localhost:8080`

---

## 🌐 Funcionalidades

- Visualización del catálogo de películas
- Gestión completa (crear, leer, actualizar, eliminar)
- Base de datos H2 en memoria para desarrollo
- API REST documentada
